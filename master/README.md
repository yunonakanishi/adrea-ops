# master/ - マスタールール

アドレア広告事業部の共通ナレッジを格納するディレクトリ。

## フォルダ構成

```
master/
├── legal/           # 法務・法令ガイド
├── company-rules/   # 社内規定・ガイドライン
└── ad-division/     # 広告事業部の組織・体制・ツール情報
```

## 資料を追加する際のフォーマット

### 受け付ける形式
| 形式 | 用途 |
|------|------|
| **HTML** | ブラウザ閲覧用ページ（GitHub Pagesで公開される） |
| **PDF** | 元資料のアーカイブ（閲覧はできるがPages上ではリンクのみ） |
| **Markdown (.md)** | テキスト中心の情報整理 |
| **画像 (PNG/JPG/SVG)** | HTML内で参照する図表・ロゴ等 |

### ファイル命名規則
- 英数字・ハイフン区切り（日本語ファイル名はURLエンコードの問題があるため避ける）
- 例: `anti-monopoly-act.html`, `brand-guidelines.pdf`

### 追加手順
1. 該当フォルダにファイルを配置
2. 必要に応じて `index.html` にリンクを追加
3. git commit & push → 自動で GitHub Pages に反映
